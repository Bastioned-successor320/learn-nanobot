# 项目一：Hello Nanobot - 你的第一个 AI 助手

## 项目简介
这是一个最简单的 Nanobot 配置示例，帮助你快速上手。

## 前置要求
- Python 3.10+
- pip install nanobot-ai
- OpenAI API Key

## 快速开始
1. 设置 API Key: `export OPENAI_API_KEY=your-key`
2. 运行: `nanobot`
3. 开始对话！

## 文件说明
- `AGENTS.md`: Agent 身份定义
- `config.json`: 基础配置

## 面试话术
> "我的第一个 Nanobot 项目是一个代码审查助手，通过配置 AGENTS.md 定义 Agent 身份，使用 OpenAI GPT-4o 作为底层模型，支持中文交互。"
